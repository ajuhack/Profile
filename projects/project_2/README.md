# Project 2: Axis Bank - Universal Underwriting Programme

## Overview
Led a bank-wide alternate-data programme feeding credit decisioning, KYC, and fraud engines at India's third-largest private bank.

## Role
Head of Alternate Data — Data Sciences (BIU)

## Duration
Dec 2022 – Dec 2023

## Key Achievements
- +25% new-to-bank acquisition efficiency
- –20% reduction in early-delinquency exposure
- Productionised 8 alternate-data assets
- Established Data Centre of Excellence
- Integrated satellite-data signals for agri-customer risk profiling

## Technologies
- Python, SQL, GCP, Tableau, GenAI, LLMs, Account Aggregator API

## Impact
- Reference case for alternate-data underwriting in Indian banking
- Expanded addressable market for rural credit
- GenAI pilots targeting 15% revenue uplift
